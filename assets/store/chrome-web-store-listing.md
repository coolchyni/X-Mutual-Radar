# Chrome Web Store Listing

## Store Name
X互关雷达

## Summary
在 x.com 页面直接标记互关和单向关注作者，并显示关注率。

## Detailed Description
X互关雷达会在 x.com 页面识别帖子作者的关注关系，并用醒目的标签直接标出来，让你不用频繁点进主页查看。

核心功能：

- 直接标记互关作者
- 识别单向关系：`我关注`、`关注我`
- 可选显示 `following / followers` 互关率
- 标签位置可配置：右上角空白区、右上角、作者信息下方
- 帖子高亮可单独开关，默认关闭
- 配置集中在扩展弹窗中，不干扰页面浏览

适合场景：

- 快速筛选时间线中的互关账号
- 识别你已关注但未互关的账号
- 识别关注了你、但你还没回关的账号
- 在浏览推文详情、时间线、用户页时快速查看关系状态

## Permissions Explanation
- `storage`：保存你的扩展开关、标签位置和显示偏好
- `tabs`：读取当前活动标签页状态，以便在弹窗中显示当前页统计
- `https://x.com/*`：仅用于在 x.com 页面注入标签和读取页面内已有的公开界面数据

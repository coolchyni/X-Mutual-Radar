# X互关雷达 隐私政策

更新日期：2026-03-29

X互关雷达重视您的隐私。本浏览器扩展的设计目标是仅在您的浏览器本地处理社交关系数据，不会向开发者服务器或任何第三方上传您的账号信息、推文内容或私密关注关系数据。

---

## 1. 扩展会读取什么

本扩展仅在 `https://x.com/*` 页面运行，并读取识别作者关系所必需的公开显示信息，包括：

- **作者账号名 (Handle)**：当前页面正在查看的账号公开 ID。
- **公开关系标识**：由 X 平台提供的状态文字（如“关注了你”）。
- **公开关注/粉丝计数**：在用户主页上公开可见的数据。
- **本地偏好设置**：您在扩展中自定义的开关状态、标签显示位置等配置。

## 2. 数据如何处理

上述所有数据仅在您的本地浏览器环境中用于以下用途：

- 判断目标账号是否与您互关。
- 在页面上直观地标记出单向关注关系。
- 计算并显示关注/粉丝互粉率。
- 根据您的偏好，在页面相应位置渲染 UI 标签和高亮效果。

## 3. 数据传输与存储

### 数据是否会被上传？
**不会。** 本扩展不会与任何外部开发者服务器通信。您的账号凭据、私密关注列表、浏览记录以及关系分析结果永远不会离开您的本地设备。

### 本地存储
扩展使用 `chrome.storage.local` API 来保存您的配置信息，包括：
- 插件开关状态
- UI 显示偏好（如标签位置）
- 功能开关（如是否显示互关率）

这些配置仅保存在您的电脑上，以便在下次使用时自动应用您的偏好。

## 4. 第三方共享

我们不会向任何第三方转售、交易或共享您的个人信息。本扩展不涉及广告投放、信用分析或数据经纪等行为。

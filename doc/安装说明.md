# X 互关雷达 (X Mutual Radar) 安装说明

欢迎使用 **X 互关雷达**！由于本插件目前暂未上架 Chrome 应用商店（Chrome Web Store），您可以通过我们提供的离线安装包（ZIP）进行手动安装。

整个过程非常简单，只需两分半钟即可完成。请按照以下详细步骤操作：

---

## 📥 第一步：下载插件

1. 前往本项目的 **GitHub Releases 页面**（[点击前往发布页](https://github.com/coolchyni/X-Mutual-Radar/releases)）。
2. 在最新的版本（Latest）下方的 **Assets** 栏中，找到以 `.zip` 结尾的压缩包文件（例如 `x-mutual-radar-v0.1.2.zip`）。
3. 点击下载该压缩包到您的电脑上。
4. 将下载的压缩包 **解压** 到一个您常用的、不会被轻易删除的文件目录（例如：`文档/X-Radar`）。
   > [!IMPORTANT]  
   > 安装后请**不要**删除或移动此文件夹，否则 Chrome 将无法读取插件文件。

---

## 🛠️ 第二步：在 Chrome 浏览器中启用

1. **打开扩展管理页面**：
   - 打开您的 Chrome 浏览器。
   - 在地址栏输入 `chrome://extensions/` 并在键盘上按下回车（Enter）键。
   - （或者：点击浏览器右上角的三个点 `⋮` -> 选择“扩展程序” -> “管理扩展程序”）。

2. **开启开发者模式**：
   - 在扩展管理页面的 **右上角**，找到名为“**开发者模式**”的开关，点击将其 **开启**。开启后，页面顶部会多出几个按钮。

3. **加载已解压的扩展程序**：
   - 点击页面左上角出现的“**加载已解压的扩展程序**”按钮。
   - 在弹出的文件夹选择窗口中，找到并选中您刚才在第一步中**解压好的文件夹**（注意：必须选中包含 `manifest.json` 文件的根目录），然后点击“选择/确定”。

4. **完成安装**：
   - 此时，**X 互关雷达** 的卡片就会出现在您的扩展列表中，并且默认处于启用状态。
   - 为了方便使用，您可以点击 Chrome 浏览器右上角地址栏旁边的 **“拼图”图标**（🧩），找到“X 互关雷达”并点击它旁边的 **大头针图标（📌 图钉）**，将其固定在浏览器的工具栏上。

---

## 🎉 第三步：开始使用与配置

1. 打开或刷新 [x.com](https://x.com) 页面。
2. 尽情体验神奇的社交关系标记功能（如遇到未生效的情况，通常刷新一下页面即可）。
3. 如需调整标签的语言、位置、字体大小或开关关注率功能，只需点击浏览器右上角固定好的**插件图标**，即可打开中文设置面板。

> [!NOTE]  
> **安全说明**：手动安装的插件与商店下载的插件具有相同的安全隔离级别。本插件基于 Chrome Manifest V3 开发，且所有的扫描和数据处理均在您的浏览器本地进行，**不会采集或上传任何隐私数据**，请放心使用。
